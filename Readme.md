# Small House For Ziyi(front-end part)
### 简介
此为微信小程序SmallHouseForZiyi的前端部分，主要设计目的是满足本人日常生活所需，例如邮件提醒的todolist，服务器监控，日程安排等等。

### 技术特点
1. 使用mpvue进行前端设计。
2. 预计采用fly库进行ajax请求。
3. 依赖vuex进行数据储存交互。
4. 利用mpvue-route进行路由管理。
5. 利用微信小程序素材库进行素材保存。
